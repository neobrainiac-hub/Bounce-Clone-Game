# 🎮 Bounce Clone Game

A simple web-based bounce game inspired by classic arcade mechanics.

## 🚀 Features

* Smooth ball movement physics
* Collision detection
* Score tracking system
* Interactive gameplay

## 🛠️ Tech Stack

* HTML

## 🎯 How to Play

* Use keyboard controls to move
* Avoid obstacles
* Keep the ball in play as long as possible
